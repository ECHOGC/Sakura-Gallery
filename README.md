# Sakura-Gallery
Sakura Gallery is a highly customizable, aesthetic, and AI-powered self-hosted private cloud gallery for NAS. Featuring a single-file frontend architecture, it integrates a silky-smooth dynamic particle engine and powerful AI visual extraction capabilities to build your exclusive digital memory archive.  Sakura Gallery 是一款极致唯美、高度可定制的轻量级 NAS 私有云相册。
